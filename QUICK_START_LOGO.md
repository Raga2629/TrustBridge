# 🚀 Quick Start - Add Your Logo

## One Simple Step

Place your logo image here:

```
trustbridge-v2/public/assets/logo.png
```

That's it! Refresh your browser and the logo will appear in the navbar.

---

## Requirements

- **File name**: `logo.png` (must be exact)
- **Format**: PNG (transparent background recommended)
- **Size**: 80-120px height (will display at 40px)

---

## What You'll See

### Before (without logo):
```
┌─────────────────────────────────────────────────┐
│  TrustBridge    Dashboard  Bookings  [👤▼]     │
└─────────────────────────────────────────────────┘
```

### After (with logo):
```
┌─────────────────────────────────────────────────┐
│  [🌉 LOGO]      Dashboard  Bookings  [👤▼]     │
└─────────────────────────────────────────────────┘
```

---

## Verify It Works

1. Place logo at: `trustbridge-v2/public/assets/logo.png`
2. Open: `http://localhost:5173`
3. Login with any account
4. Check navbar - logo should be on the LEFT
