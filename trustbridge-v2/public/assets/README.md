# Assets Folder

## Logo Image

Please place your TrustBridge logo image in this folder with the name:

**logo.png**

### Requirements:
- File name: `logo.png` (exactly)
- Recommended format: PNG with transparent background
- Recommended height: 80-120px (will be displayed at 40px height)
- The image will maintain its aspect ratio automatically

### Current Status:
The navbar is configured to use `/assets/logo.png`. If the image is not found, it will fallback to text "TrustBridge".

### How to Add:
1. Save your logo image as `logo.png`
2. Place it in this folder: `trustbridge-v2/public/assets/`
3. Refresh your browser to see the logo in the navbar
